# 🤖 Telegram Portfolio Bot

Бот-визитка Python-разработчика. Показывает услуги, цены и контакты.

## 📋 Функционал

- Меню с кнопками
- Список услуг
- Прайс-лист
- Контактная информация

## 🛠 Технологии

- Python 3.11
- pyTelegramBotAPI
- python-dotenv

## 🚀 Запуск

1. Клонируй репозиторий
2. Установи зависимости:
   pip install pyTelegramBotAPI python-dotenv
3. Создай файл .env:
   BOT_TOKEN=твой_токен
4. Запусти:
   python tg_bot.py
